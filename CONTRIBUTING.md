## Submitting Changes

1. Create a branch: `git checkout -b fix/your-fix`
2. Make changes and test locally
3. Commit: `git commit -m "fix: description"`
4. Push: `git push origin fix/your-fix`
5. Open a Pull Request

## Commit Messages

Use prefixes:

- `fix:` - bug fixes
- `feat:` - new features
- `docs:` - documentation
- `content:` - content updates
